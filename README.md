## Overview

calcolas App provides basic arithmetic operations with a clean and responsive user interface.  
The application focuses on correctness, maintainability, and user experience.

---

## Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Decimal number support
- Clear / Reset functionality
- Responsive layout for different phone screen sizes
- Input validation
- Division-by-zero protection

---

## Tech Stack

- Language: Java 
- UI: XML Layouts
- IDE: Android Studio
- Build System: Gradle
- Platform: Android SDK

---

## Error Handling

- Prevents invalid numeric input
- Handles division by zero safely
- Avoids application crashes due to malformed expressions
